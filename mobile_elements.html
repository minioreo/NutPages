移动端常用元素

罗列移动端常见的容器及元素

容器

各种容器，用于将各种元素组成一个逻辑单元。

ActionBar

  • 用途，页面标题兼操作栏，可包含功能按钮。

  • 代码示例:

    <div class="header">
    <a data-role="button">左侧按钮</a>
        <h1>标题</h1>
    <a data-role="button">右侧按钮</a>
</div>

  • 属性:无

Tab Nav

  • 用途：tab导航，可位于页面上方或下方

  • 代码示例:

    <div data-role="navbar">
    <ul>
        <li><a href="#" class="ui-btn-active">Tab1</a></li>
        <li><a href="#">Tab2</a></li>
        <li><a href="#">Tab3</a></li>
    </ul>
</div>

  • 属性：无

Sliding Menu

  • 用途：可以从左侧或右侧滑出的菜单

  • 代码示例

    <nav class="sliding-menu-left" >
    <h3>Menu</h3>
    <a href="#">用户管理</a>
    <a href="#">组织管理</a>
    <a href="#">菜单管理</a>
    <a href="#">权限管理</a>
</nav>

  • 属性:无

Control Group

  • 用途：用于组合多个button，形成操作栏，可竖排或者横排。

  • 代码示例： 

    <div data-role="controlgroup" data-type="horizontal">
    <a href="#" data-role="button">new</a>
    <a href="#" data-role="button">edit</a>
    <a href="#" data-role="button">delete</a>
</div>

  • 属性

    • data-type:横向/纵向

Collapsibles

  • 用途：可折叠的区域，可点击标题栏折叠或展开区域

  • 代码示例:

    <div data-role="collapsible">
    <h4>标题</h4>
    <p>content area...</p>
</div>
  • 属性:
    • data-toggle:是否折叠

Panel

  • 用途：包含标题栏的容器

  • 代码示例：

    <div class="panel panel-default">
    <div class="panel-heading">
        <h3 class="panel-title">Panel title</h3>
    </div>
    <div class="panel-body">
Panel content
</div>

  • 属性

    • data-collapsed-icon：分组折叠时的图标
    • data-expanded-icon：分组展开时的图标

Form

  • 用途：标准的html表单，采集用户输入

  • 代码示例：

    <form role="form">
    <div class="form-group">
        <label for="exampleInputEmail1">Email address</label>
        <input type="email" class="form-control" id="exampleInputEmail1" placeholder="Enter email">
    </div>
    <div class="form-group">
        <label for="exampleInputPassword1">Password</label>
        <input type="password" class="form-control" id="exampleInputPassword1" placeholder="Password">
    </div>
    <div class="form-group">
        <label for="exampleInputFile">File input</label>
        <input type="file" id="exampleInputFile">
        <p class="help-block">Example block-level help text here.</p>
    </div>
    <div class="checkbox">
        <label>
            <input type="checkbox"> Check me out
        </label>
    </div>
    <button type="submit" class="btn btn-default">Submit</button>
</form>
  • 属性：无

页面元素

移动端页面基本元素

Text Input

  • 用途：文本输入框，可能需要带一些输入校验，错误提示。

  • 代码示例:

    <input type="text" class="form-control" placeholder="Text input">

  • 属性：

    • type:类型：text，password等
    • placeholder:占位符
    • disabled:是否禁用
    • 一些预设的样式，用于表示输入框状态，如success，error，warning等

CheckBox

  • 用途：多选框

  • 代码示例：

    <input type="checkbox" >
  • 属性:
    • checked:是否选中

radio button

  • 用途：单选,移动平台一般会实现为Toggle Button

  • 代码示例：

    <div class="radio">
    <label>
    <input type="radio" name="optionsRadios" value="option1" checked>
    option1
    </label>
</div>
<div class="radio">
    <label>
    <input type="radio" name="optionsRadios" value="option2">
    option2
    </label>
</div>
  • 属性：无

date time picker

  • 用途：选择日期

  • 代码示例：

    <input type="datetime">

  • 属性：

    • type:date/month/week/time/datetime/datetime-local

Select

  • 用途：单选控件

  • 代码示例：

    <select>
    <option>选项1</option>
    <option>选项2</option>
</select>
  • 属性：无

Button

  • 用途：按钮

  • 代码示例:

    <button type="button" class="btn">Click</button>

  • 属性:

    • disabled:是否禁用
    • 样式

Image

  • 用途：显示图片

  • 代码示例：

    <img src="..." alt="..." class="img-rounded">

  • 属性：

    • 一些预设样式

Icons

  • 用途：小图标，用于形象的表明操作意图。交流行的方式是使用web font实现。

  • 代码示例：

    <span class="glyphicon glyphicon-search"></span>

  • 属性：无

Modal Dialog

  • 用途：弹出窗口，用于提示信息或收集少量用户输入
  • 代码示例:根据使用的ui库不同而不同，一般是页面用带样式的div定义对话框ui，使用ui库js方法弹出。

Animation

  • 用途：各种动画，应用于元素显隐，页面切换或组件切换时的效果。
  • 代码示例：移动端可使用css3动画实现或其他javascript库
